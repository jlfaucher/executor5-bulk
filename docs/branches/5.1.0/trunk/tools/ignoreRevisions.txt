- this file lists revision numbers that should be ignored by updateEntityValures.rex
- e.g. a revision number where the copyright statement got updated and saved does not
- reflect a true revision of the content of the book such that such a revision should
- get ignored, hence this file

- syntax of the file: - empty lines and lines starting with dash (-) or hash (#) get
-                       ignored and can be used for comments
-                     - any line containing a different entry is taken as a revision string
-                       (a number for svn some string for git or other revision control
-                       systems) that should be ignored

Welcome

Welcome to the Open Object Rexx Project build machine. This machine will allow you to build the current Subversion Revision of ooRexx using an automated system hosted on this web site.

The Build Machine is hosted using Fedora Core 6 with a combination of VMware Server 1.0.3 and Linux KVM (on another server). It hosts a number of virtual machine guest operating systems which will be used to perform the actual builds.

The button to the left labeled Build ooRexx will take you to the selection page for building the ooRexx packages.

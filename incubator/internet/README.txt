memcache.cls

   This class provides access to a memcached daemon running on a network
   server. The memcached daemon is a hash-based mechanism for storing data
   in memory on the server.

mime.cls

   This is a class wrapper for a mime object.

smtp.cls

   This is a class for creating and sending SMTP mail. It uses the mime.cls
   and the strmsock.cls to perform its function.

strmsock.cls

   This set of classes encapsulate the TCP/IP rxsock package for stream sockets.
   There is a base class. a stream client class and a stream server class. This
   make using the rxsock package much easier as it hides most of the
   complexity of using stream sockets.

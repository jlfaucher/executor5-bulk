head.rex

   This is a simple clone of the Unix head utility. It displays the first n
   lines of a text file. By default, n = 10.

tail.rex

   This is a simple clone of the Unix tail utility. It displays the last n
   lines of a text file. By default, n = 10.

tee.rex

   This is a simple clone of the Unix tee utility. It displays text piped to it
   via stdout. It also takes that output and either appends it to a file or
   replaces the contents of a file with that text.

touch.rex

   Sets the modified timestamp of a set of files to the current date/time.
   It can also recurse into child subdirectories.

isa.cls

   The isa.cls provides a set of public methods and routines that can be
   used to determine the type of class a method is a part of.

orxutils.cls

   This class provides some useful utility methods and routines.

replace.rex

   This script will replace the specified target search text with the specified
   replacement text. The script starts looking in files in the current
   subdirectory and recurses through all the child subdirectories processing
   each file it finds. The search and replacement text are specified within
   the script, not on the command line.

text2rexx.rex

   This script takes the contents of a text file, converts it into a Rexx
   assignment statement, and then writes that assignment statement to an
   output file.
